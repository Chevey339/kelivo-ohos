## 📖 项目简介

[Kelivo](https://github.com/Chevey339/kelivo) 的OpenHarmony 版本，有些功能（像TTS功能或代码块高亮）等一些次要功能因为鸿蒙的一些Flutter依赖支持问题暂未实现。

## 🔧 OpenHarmony 兼容性

本项目针对 OpenHarmony 系统进行了适配，确保在鸿蒙设备上可以稳定运行。
